# Foobar

foobar is a python library for dealing with word plurazlization

## Installition

use package manager [pip](https://google.com) to install foobar.

```
pip install foobar
```

## Usage

```
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
